create table leads ( id text primary key, ts bigint, nome text, wpp text, email text, idade text, para text, obj text, nivel text, inicio text, periodo text, motivacao text, origem text, score text, etapa text default 'Novo Lead', proxima_acao text default 'Primeiro contato', notas jsonb default '[]', data text, hora text, fonte text, escola text, created_at timestamptz default now() ); alter table leads enable row level security; create policy "acesso_publico" on leads for all using (true) with check (true);

📖 Playbook
Como a Bolsão atende
Guia prático do time. Conduza qualquer lead com leveza e consistência.
≤10min
1º contato
3
etapas
💬
Diretrizes gerais
Como se comunicar e criar conexão
😊
Tom leve e próximo
Fala como gente. Usa o nome, mostra interesse genuíno.
🎯
Personalização sempre
Leu o formulário? Mencione. Ela percebe quando você se preparou.
👂
Ouvir mais do que falar
Deixe a pessoa contar a história antes de apresentar qualquer coisa.
⚡
Velocidade gera confiança
Lead sem resposta em 10 minutos já está comparando com a concorrência.
🧭
Conduza, não empurre
"Quando você pode vir conhecer?" é melhor do que esperar a pessoa decidir.
📋
Registre tudo
Depois de cada contato, atualiza o CRM. Memória curta perde venda.
🗺️
3 etapas do atendimento
1
Contato inicial
Confirmação e agendamento
Confirmar interesse, entender o perfil e marcar a visita.
Como iniciar
Chame pelo nome. Sempre.
Mencione o objetivo dela
2 frases mostrando que leu o formulário
Ao agendar
Ofereça 2–3 opções de horário
"Sem compromisso, só pra conhecer"
Mande o endereço em seguida
2
Reunião
Apresentação e proposta
Entender a fundo e apresentar a escola com contexto.
Perguntas-chave
"Qual sua maior dificuldade com tecnologia?"
"Como seria sua vida em 6 meses resolvendo isso?"
Apresentar valor
Conecte ao objetivo dela
Fale de resultado, não de conteúdo
3
Integração
Boas-vindas e início
Gerar segurança e garantir começo empolgante.
Segurança
"Qualquer dúvida, chama no WhatsApp"
Apresente o professor pelo nome
CRM
Mover para "Matriculado" imediatamente
Registrar turma e data de início
⚖️
Regras do atendimento
1
Leia o formulário antes de chamar. Atendimento genérico não converte.
2
Nunca deixe lead sem resposta. 10 minutos é o máximo.
3
Sempre defina próxima ação. Toda conversa termina com um próximo passo.
4
Preço não pelo WhatsApp. Leve para visita. Presencial converte mais.
5
Registre tudo no CRM. Se não está registrado, não aconteceu.
6
Tom leve sempre. Essa é a identidade da Bolsão.
